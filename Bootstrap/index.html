Templating

Templating allows us to define a preset "pattern" for a webpage, that we can dynamically modify.

For example, we could define a single "Search" template that displays all the results for a given search term. We don't know what the term is or how many results ther are ahead of time. The webpage is created "on the fly."

Available tools to help us with templating:

  • EJS = Embedded JavaScript
    • Uses ACTUAL JS syntax!
  • Handlebars
  • Jade
  • Pug
  • Nunjucks
  • And More!
